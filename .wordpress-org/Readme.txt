== Description ==
Módulo de cita online donde los pacientes podrán pedir una cita desde la página web.


== Frequently Asked Questions ==
= ¿Cómo consigo la APIKEY? =
La APIKEY se proporciona al centro que tenga contratado el módulo de Cita Online de Infomed Software.
Mas información aquí: https://www.grupoinfomed.es/software-sanitario/cita-online-g5/


== Installation ==

1. Activa el plugin `Cita Online`.
2. En el menú nos aparecerá `Cita Online`.
3. Establecemos la APIKEY proporcionada por Infomed Software.
4. Añade `[cita-online]` en la página donde quieras mostrar el formulario.
